		Instrucciones para compilar cada clase

1.- Abrir una terminal en la carpeta "src"
2.- Para compilar cualquier clase escribir en la terminal el siguiente comando:

	javac shapesSVG/<Nombre_de_la_clase>.java

3.- Para ejecutar cualquier clase escribir en la terminal el siguiente comando:

	java shapesSVG/<Nombre_de_la_clase>.java


		Para compilar y ejecutar Main.java

1.- Abrir una terminal en la carpeta "src"
2.- Para compilar escribir el siguiente comando

	javac main/Main.java

3.- Para ejecutar escribir el siguiente comando

	java main/Main.java

esto imprimirá el código html y SVG en la terminal

4.- Para imprimir el código en un archivo .html escribir el siguiente comando:

	java main/Main.java > Main.html

esto imprimirá el código en un archivo llamado Main.html, se le puede poner cualquier otro nombre al archivo

5.- El archivo Main.html se creará dentro de la carpeta "src", dar doble click en el archivo lo abrirá en una pestaña de navegador y éste mostrará un ejemplo de cada una de las clases.
